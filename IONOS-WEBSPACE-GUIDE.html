Troubleshooting

Problem: Website doesn't load
  • Solution: Wait 10-15 minutes. DNS changes can take time
  • Try clearing your browser cache (Ctrl+Shift+Delete)
  • Try viewing in an incognito/private window
Problem: Page looks broken or unstyled
  • Solution: Check that the assets folder was uploaded correctly
  • Make sure both CSS and JS files are in the assets folder
  • File names are case-sensitive and must match exactly
Problem: Legal pages show 404 error
  • Solution: Check that the legal folder was created with the correct name (all lowercase)
  • Verify all 4 HTML files are inside the legal folder
Problem: Can't find File Manager
  • Solution: Look under "Websites & Domains" in your IONOS dashboard
  • Or search for "WebSpace" or "FTP" in the IONOS help search
  • You can also use FTP software like FileZilla if you prefer
